# GMAIL-Auto-Clean-n8n

**📌 Gmail Auto-Clean Automation using n8n**

This project is an automated email management workflow built using n8n to keep the Gmail inbox clean and organized.

The workflow runs on a scheduled basis and intelligently filters incoming emails based on predefined conditions. It identifies unread emails from categories like Promotions, Social, and Updates, and automatically labels them under a custom tag (Auto-Clean) for easy review and cleanup.

This approach reduces manual effort, improves inbox hygiene, and ensures that important emails remain untouched while low-priority emails are grouped separately.

**⚙️ Key Features**
Automated email processing using n8n
Filters emails based on:
Unread status
Gmail categories (Promotions, Social, Updates)
Applies custom label (Auto-Clean)
Prevents reprocessing of already labeled emails
Scheduled execution (daily at 9 AM)

**🛠️ Tech Stack**
n8n (workflow automation)
Gmail API
🎯 Use Case

Managing a cluttered inbox by automatically organizing non-essential emails, allowing users to focus only on important communications.

**🚀 Outcome**
Reduced manual email sorting effort
Improved productivity
Structured and organized inbox
